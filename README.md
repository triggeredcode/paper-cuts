# Paper Cuts

A deceptively simple problem about cutting a page into pieces — that opens into surprisingly deep design questions.

This is a small two-piece site: an **essay** that names the problem and surfaces the design questions, plus an **interactive playground** where you can actually cut a page and watch the regions emerge.

## Pages

- **[Homepage](./index.html)** — landing page with both entry points
- **[Essay](./blog.html)** — the article (~10 min read) on PSLG, DCEL, T-junctions, and 11 design questions you have to answer before you write any code
- **[Game](./game.html)** — interactive paper-cutting tool. Click two points, see the cut. Snap to edges. Scatter the pieces.

All three are static HTML files — no build step, no dependencies. Open `index.html` in a browser to start.
